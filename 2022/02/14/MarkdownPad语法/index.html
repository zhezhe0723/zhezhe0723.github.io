MarkdownPad语法

发表于 2022-02-14 更新于 2022-02-22 阅读次数： 本文字数： 1.8k 阅读时长 ≈ 2 分钟

前言

我之前记笔记一直用OneNote，但是年前心血来潮，就自己建了博客，想着把自己的学习心得发到博客上。但是博客算是完工了，但是写文章又成了一大难题，就自己在网上查找写博客的编辑器，发现用Markdown Pad来编写文章很方便。但是刚接触这个编辑器，其中的语法看了几次记不住，于是准备写一下Markdown Pad的写作语法。以供自己写作的时候还要在网上找（麻烦….）。

语法介绍

段落文本

换行：直接敲回车键即可

空格：

   &emsp;//两个空格大小，也就是一个中文字符
   &ensp;//一个空格大小，也就是半个中文字符
   &nbsp;//1/2个空格大小，也就是1/4个中文字
   （但是现在这个功能好像不可以）

文本：

*斜体文本*  _斜体文本_  //两种方式都可
**加粗文本**  __加粗文本__  //两种方式都可
***斜体加粗***  ___斜体加粗___  //两种方式都可
<u>下划线</u>
^上表^  //具体以指数公式展示
~下标~  //具体以水的化学式展示

分割线：分割线的语法只需要三个 * 号或者在选项卡中添加水平标尺，快捷键 CTRL+R

标题

标题有三种设置方式：①手动添加；②快捷键添加；③选项卡添加；

①手动添加 

                   #一级标题的内容#
                  ##二级标题的内容##
                 ###三级标题的内容###
    以此类推

②快捷键添加
直接在键盘上输入ctrl+num,这里的num是指标题的级数，例如：设置一级标题 ctrl+1以此类推
③选项卡添加（可以忽略不计）

列表

有序列表

有序列表，顾名思义，即输入的内容按序号进行标识排列。
①手动添加
在输入的内容前，添加num. + 空格 ，接着按 回车 ，即可添加num+1条内容
②快捷键添加
直接在键盘上输入Ctrl+Shift+O，输入第一条内容后，按 回车 ，即可添加第二条内容。注意快捷键输入的是 字母O ，而不是 数字0。

无序列表

&emsp;&emsp;无序列表，顾名思义，即输入的内容不按序号进行排列。
①手动添加
在输入的内容前，添加-/* + 空格 ，接着按回车 ，即可添加下一条内容。
②快捷键添加
直接在键盘上输入Ctrl+U，输入第一条内容后，按回车 ，即可添加第二条内容。

引用

&emsp;&emsp;引用可以将一段文字进行单独的标识，
①手动添加
在需要标注引用的内容前，输入>，即可将此内容标记为引用类型。
②快捷键添加
直接在键盘上输入Ctrl+Q。

图片与超链接

图片

  1. 网上的图片，添加到正在编辑的文本中进行展示。此时我们便需要用到插入图片的语法。![name](url) name 是添加图片的文字描述，自己添加即可，url 是添加图片的网址
  2. 本地的图片，使用相对路径插入图片。比如你把一个叫做1.png的图片和*.md文件放在一起，那么你就可以用这种方式插入图片：![](1.png)
    如果不想放在同一层级,那么就可以这样插入:![](foldername/1.png) 表示引用同层级一个叫做”foldername”的文件夹中的1.png图片,以此类推

    超链接

    &emsp;&emsp;超链接和图片的添加方式只差一个!号

图片为：![]()

链接为：[]()

表格

  • 表格的话直接用HTML写就可以

      <table>
    <tr>
        <td>姓名</td>
        <td>性别</td>
    </tr>
    <tr>
        <td>李斯</td>
        <td>男</td>
    </tr>
  </table>

  • 代码块：前后添加 ``` 三个即可
